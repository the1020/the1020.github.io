RefлҠ” DOM л…ёл“ңлӮҳ лҰ¬м•ЎнҠё м—ҳлҰ¬лЁјнҠём—җ м ‘к·јн•ҳлҠ” л°©лІ•мқ„ м ңкіөн•ңлӢӨ. HTML мҡ”мҶҢм—җ нҸ¬м»ӨмҠӨлҘј мЈјкұ°лӮҳ м• лӢҲл©”мқҙм…ҳ мҲҳн–үмқ„ мң„н•ҙ мӮ¬мҡ©н•ңлӢӨ. Ref м „лӢ¬н•ҳкё°лҠ” м»ҙнҸ¬л„ҢнҠёк°Җ мһҗмӢқ м»ҙнҸ¬л„ҢнҠёмқҳ refлҘј мһҗмӢ мқҳ refлЎңм„ң мҷёл¶Җм—җ л…ём¶ңмӢңнӮӨкІҢ н•©лӢҲлӢӨ.

л¶ҖлӘЁ м»ҙнҸ¬л„ҢнҠёмқё Login м»ҙнҸ¬л„ҢнҠём—җм„ң м„ м–ён•ң refлҘј мһҗмӢқ м»ҙнҸ¬л„ҢнҠёмқё Input м»ҙнҸ¬л„ҢнҠёлЎң к°Ғк°Ғ м „лӢ¬н•ңлӢӨ. Input м»ҙнҸ¬л„ҢнҠём—җм„ң React.forwardRefлЎң м»ҙнҸ¬л„ҢнҠёлҘј мғқм„ұн•ҳмҳҖкі  React.forwardRefлҠ” propsмҷҖ ref нҢҢлқјлҜён„°лҘј л°ӣм•„ React л…ёл“ңлҘј л°ҳнҷҳн•ҳлҠ” л ҢлҚ”л§Ғ н•ЁмҲҳлҘј л°ӣлҠ”лӢӨ. мқҙлҹ° л°©лІ•мңјлЎң Inputмқ„ мӮ¬мҡ©н•ҳлҠ” м»ҙнҸ¬л„ҢнҠёл“ӨмқҖ input DOM л…ёл“ңм—җ лҢҖн•ң м°ёмЎ°лҘј к°Җм ёмҳ¬ мҲҳ мһҲкі , н•„мҡ”н•ң кІҪмҡ° DOM inputмқ„ м§Ғм ‘ мӮ¬мҡ©н•ҳлҠ” кІғмІҳлҹј м ‘к·јн•  мҲҳ мһҲлӢӨ.

вҡ пёҸ л‘җ лІҲм§ё ref мқёмһҗлҠ” React.forwardRefмҷҖ к°ҷмқҙ нҳём¶ңлҗң м»ҙнҸ¬л„ҢнҠёлҘј м •мқҳн–Ҳмқ„ л•Ңм—җл§Ң мғқм„ұлҗ©лӢҲлӢӨ. мқјл°ҳ н•ЁмҲҳлӮҳ нҒҙлһҳмҠӨ м»ҙнҸ¬л„ҢнҠёлҠ” ref мқёмһҗлҘј л°ӣм§ҖлҸ„ м•Ҡкі  propsм—җм„ң мӮ¬мҡ©н•  мҲҳлҸ„ м—ҶмҠөлӢҲлӢӨ.

useImperativeHandle

Loginм—җм„ң Inputмқҳ refм—җ м ‘к·јн•ҳкё° мң„н•ҙм„ңлҠ” useImperativeHandleмқ„ мӮ¬мҡ©н•ңлӢӨ. мң„ Login м»ҙнҸ¬л„ҢнҠём—җ лІ„нҠј 2к°ң(focusEmail, focusPassword)мҷҖ к°Ғ лІ„нҠјмқ„ нҒҙлҰӯн•ҳл©ҙ мқҙл©”мқј лҳҗлҠ” л№„л°ҖлІҲнҳё Inputм—җ focusлҘј мЈјлҠ” н•ЁмҲҳ(loginInputFocusHandler, passwordInputFocusHandler)лҘј м¶”к°Җн•ҳмҳҖлӢӨ.

//рҹ“„ Login.js
const Login = () => {
  const emailRef = useRef();
  const passwordRef = useRef();

  const loginInputFocusHandler = () => {
    emailRef.current.focus();
  }

  const passwordInputFocusHandler = () => {
    passwordRef.current.focus();
  }

  return (
    <>
      <Input type="email" ref={emailRef} />
      <Input type="password" ref={passwordRef} />
      <button id="focusEmail" onClick={loginInputFocusHandler}>Email Focus</button>
      <button id="focusPassword" onClick={passwordInputFocusHandler}>Password Focus</button>
    </>
  );

  //рҹ“„ Input.js
  const Input = React.forwardRef((props, ref) => {
    const inputRef = useRef();

    const activate = () => {
        inputRef.current.focus();
    };

    useImperativeHandle(ref, () => {
        return { focus: activate };
    });

    return <input type={props.type} ref={inputRef}>
  });
};

мқҙл©”мқј Inputмқҳ нҸ¬м»ӨмҠӨлҘј мң„н•ң лІ„нҠјмқ„ нҒҙлҰӯн•ҳл©ҙ emailRef.current.focus();к°Җ мӢӨн–үлҗҳкі  мқҙ л•Ң focus()лҠ” Inputм—җм„ң useImperativeHandle лӮҙм—җм„ң л°ҳнҷҳлҗң к°қмІҙмқҳ focusлқјлҠ” мҡ”мҶҢмқҳ н•ЁмҲҳмқҙлӢӨ. focusлҠ” activateлқјлҠ” н•ЁмҲҳлҘј лӢҙкі  мһҲкё° л•Ңл¬ём—җ мөңмў…м ҒмңјлЎң activate()к°Җ мӢӨн–үлҗңлӢӨ.

мҰү, focusEmail лІ„нҠјмқ„ нҒҙлҰӯн•ҳл©ҙ Inputмқҳ activateк°Җ мӢӨн–үлҗҳл©° input DOM мҡ”мҶҢм—җ нҸ¬м»ӨмҠӨк°Җ мһЎнһҢлӢӨ.

лӢӨлҘё н”„лЎңм қнҠём—җм„ң мӢӨм ң мӮ¬мҡ©н•ң forwardRefмҷҖ useImperativeHandle м•Ңм•„ліҙкё°

TocklerлҠ” м• н”ҢлҰ¬мјҖмқҙм…ҳмқҳ мқҙмҡ© мӢңк°„мқ„ м¶”м Ғн•ҳлҠ” м„ңл№„мҠӨлЎң мқјл үнҠёлЎ кіј лҰ¬м•ЎнҠё, нғҖмһ…мҠӨнҒ¬лҰҪнҠёлЎң к°ңл°ңлҗҳм—ҲлӢӨ.

л©”мқё нҷ”л©ҙ мғҒлӢЁм—җлҠ” кё°к°„мқ„ м„ нғқн•ҳкі  к·ё кё°к°„л§ҢнҒјмқҳ мқҙмҡ© мӢңк°„мқ„ м°ЁнҠёлЎң ліҙм—¬мЈјлҠ” кё°лҠҘмқҙ мһҲлӢӨ. DataRangeInputм—җм„ң кё°к°„мқ„ м„ нғқн•ҳлҠ” input мҡ”мҶҢмқҳ к°’мқҙ ліҖкІҪлҗҳл©ҙ datepickerмқҳ м„ нғқлҗң лӮ м§ң к°’лҸ„ ліҖкІҪн•ҳлҠ” лӮҙмҡ©мқҙлӢӨ.

м—¬кё°м„ң мӮ¬мҡ©н•ҳлҠ” datePicker м»ҙнҸ¬л„ҢнҠёмқҳ мҶҢмҠӨлҘј мӮҙнҺҙліҙл©ҙ refлҘј м–ҙл–»кІҢ мӮ¬мҡ©н•ҳлҠ”м§Җ м•Ң мҲҳ мһҲлӢӨ. (мӢӨм ң н”„лЎңк·ёлһЁм—җм„ңлҠ” input мҡ”мҶҢмқҳ к°’мқ„ мһ„мқҳлЎң ліҖкІҪн•  мҲҳ м—ҶлӢӨ. лӮҳмӨ‘м—җ мҶҢмҠӨ л¶„м„қмқ„ н•ҳкІҢ лҗҳл©ҙ мһҗм„ёнһҲ м•Ңм•„лҙҗм•јкІ лӢӨ.)

м•„лһҳ мҪ”л“ңлҠ” нғҖмһ…мҠӨнҒ¬лҰҪнҠё л¬ёлІ•мқҖ мғқлһөн•ҳкі  н•ҙлӢ№ лӮҙмҡ©мқ„ м„ӨлӘ…н•ҳкё° мң„н•ң мҪ”л“ңл§Ңмқ„ к°Җм ёмҳЁ кІғмқҙлӢӨ. мӢӨм ң мҪ”л“ңлҠ” м•„лһҳ к№ғн—ҷ л§ҒнҒ¬лҘј м°ёкі н•ҳл©ҙ лҗңлӢӨ.
